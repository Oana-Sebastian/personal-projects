## 3. ESP32 BLE Macro Keyboard

**Stack:** C, ESP-IDF, FreeRTOS, BLE HID, CMake  
**Context:** A hardware-software integration project exploring Embedded Systems and low-level communication.  
**Goal:** To engineer custom firmware that translates physical hardware inputs into wireless OS-level commands via Bluetooth.

### Objectives:
- Implement the HID over GATT profile for wireless media and keystroke control.
- Interface GPIO and ADC peripherals to process digital button matrices and analog signals.
- Orchestrate concurrent task management using FreeRTOS for non-blocking hardware debouncing.

[Link to Project Repository](https://github.com/Oana-Sebastian/ESP32-BLE-MacroKeyboard.git)
